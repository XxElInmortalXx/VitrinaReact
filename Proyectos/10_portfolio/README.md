## Ejecutar localmente

Clona el proyecto

```bash
  git clone https://github.com/XxElInmortalXx/VitrinaReact/tree/main/Proyectos/09_portfolio
```

Ir al directorio del proyecto

```bash
  cd 09_portfolio
```

Instalar las dependencias

```bash
  npm install
```

Iniciar el proyecto como desarrollador

```bash
  npm run dev
```
