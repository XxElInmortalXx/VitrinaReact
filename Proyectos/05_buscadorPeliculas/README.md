## Ejecutar Localmente

Clona el proyecto

```bash
  git clone https://link-to-project
```

Ir al directorio del proyecto

```bash
  cd my-project
```

Instalar las dependencias

```bash
  npm install
```

Iniciar el proyecto como desarrollador

```bash
  npm run dev
```
